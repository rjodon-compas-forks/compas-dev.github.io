hpc

This package provides GPU-accelerated or JIT-compiled versions of many geometry, numerical and topological functions and algorithms. The package id built around Numba, PyCuda and PyOpenCL.

Warning

The functionality of this package is experimental and subject to frequent change. For now, don’t use it for anything important :)
