geometry

This package provides functionality for working with geometry outside independent of CAD software.

Algorithms

Compute mesh flatness per face.

Planarize the faces of a mesh using ShapeOp.

Smooth a mesh by moving every free vertex to the centroid of its neighbours.

Parallelise the edges of a network to given target vectors.

Smooth a network by moving each vertex to the centroid of its neighbours.

oriented_bounding_box_numpy

oriented_bounding_box_xy_numpy

Compute the oriented minimum bounding box of set of points in the XY plane.

Planarise a set of connected faces.

scalarfield_contours_numpy

Compute the contour lines of a scalarfield.

smooth_area

Smooth a set of connected vertices by moving each vertex to the centroid of the surrounding faces, weighted by the area of the face.

smooth_centroid

Smooth a connected set of vertices by moving each vertex to the centroid of its neighbours.

smooth_centerofmass

Smooth a connected set of vertices by moving each vertex to the center of mass of the polygon formed by the neighbouring vertices.
