compas_knit

compas_knit
  • author: Mariana Popescu
  • status: private
  • version: 0.1.0
https://github.com/compas-dev/compas_knit

Knitting package for the compas framework.

Knitting offers the possibility of creating 3D geometries, including non-developable surfaces, within a single piece of fabric without the necessity of tailoring or stitching. To create a CNC-knitted fabric, a knitting pattern is needed in the form of 2D line-by-line instructions.

The compass_knit package offers the possibility of creating knitting patterns to be used in the producing of CNC knits for a given 3D model.A quad mesh model is needed for generating the patterns. The desired knitting parameters need to be defined for each mesh. These are the loop width, the loop height, and the desired knitting direction. Through the use of a network data structure the package helps create topologies which represent a knitting pattern with increases, decreases and short-rows. Once the topologies are constructed, they can be translated into 2D pixel grids to be transferred to a knitting machine. Conversely, they can be further used for simulation of knit textile behaviors.

Currently the package includes functions for contouring and generating patterns for given geometries as single thickness textile. The package will be extended to generate patterns for spacer fabrics.
