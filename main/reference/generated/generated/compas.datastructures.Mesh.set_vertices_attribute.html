Mesh.set_vertices_attribute

Mesh.set_vertices_attribute(name, value, keys=None)

Set one attribute of multiple vertices.

Parameters

  • name (str) – The name of the attribute.

  • value (object) – The value of the attribute.

  • keys (iterable (None)) – A list of vertex identifiers. Defaults to all vertices.
