Point

class compas.geometry.Point(x, y, z=0.0)[source]

A three-dimensional location in space.

Parameters

  • x (float) – The X coordinate of the point.

  • y (float) – The Y coordinate of the point.

  • z (float, optional) – The Z coordinate of the point. Default is 0.0.

x

float – The X coordinate of the point.

y

float – The Y coordinate of the point.

z

float – The Z coordinate of the point.

Examples

>>> p1 = Point([1, 2, 3])
>>> p2 = Point([4, 5, 6])

>>> p1.x
1.0
>>> p1[0]
1.0
>>> p1[5]
1.0
>>> p1[-3]
1.0
>>> p1[-6]
1.0

>>> p1 + p2
[5.0, 7.0, 9.0]
>>> p1 + [4, 5, 6]
[5.0, 7.0, 9.0]
>>> p1 * 2
[2.0, 4.0, 6.0]
>>> p1 ** 2
[1.0, 4.0, 9.0]
>>> p1
[1.0, 2.0, 3.0]

>>> p1 += p2
>>> p1 *= 2
>>> p1 **= 2
>>> p1
[100.0, 196.0, 324.0]

Notes

A Point object supports direct access to its xyz coordinates through the dot notation, as well list-style access using indices. Indexed access is implemented such that the Point behaves like a circular list.

For more info, see 1.
