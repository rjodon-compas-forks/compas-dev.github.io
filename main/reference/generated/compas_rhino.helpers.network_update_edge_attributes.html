network_update_edge_attributes

compas_rhino.helpers.network_update_edge_attributes(network, keys, names=None)[source]

Update the attributes of the edges of a network.

Parameters

  • network (compas.datastructures.Network) – A network object.

  • keys (tuple, list) – The keys of the edges to update.

  • names (tuple, list (None)) – The names of the atrtibutes to update. Default is to update all attributes.

Returns

bool – True if the update was successful. False otherwise.
